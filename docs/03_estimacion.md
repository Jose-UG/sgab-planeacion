# Estimación del Proyecto

## Suposiciones de la Estimación

- Equipo de 2 desarrolladores.
- Jornada parcial de 4 horas diarias por desarrollador.
- Dedicación de 5 días a la semana.
- Complejidad media del proyecto.
- Tecnología web estándar (frontend + backend + base de datos).

---

## Descomposición funcional (módulos)

1. Gestión de Usuarios (Administrador)
2. Gestión de Materias (Administrador)
3. Gestión de Alumnos (Administrador)
4. Gestión de Calificaciones (Docente)
5. Autenticación y Control de Acceso (Todos)

---

## Estimación de Esfuerzo (horas/persona)

| Módulo                            | Esfuerzo (horas) |
|-----------------------------------|------------------|
| Gestión de Usuarios               | 20 h             |
| Gestión de Materias               | 20 h             |
| Gestión de Alumnos                | 30 h             |
| Gestión de Calificaciones         | 30 h             |
| Autenticación y Control de Acceso | 50 h             |
| Pruebas y Ajustes                 | 40 h             |
| **Total**                         | **190 h**        |

---

## Nivel de Incertidumbre

La estimación tiene un nivel de incertidumbre del 20%, debido a factores como:

- Posibles cambios en los requerimentos.
- Dificultades técnicas imprevistas.
- Ajustes en seguridad.

El esfuerzo total estimado con incertidumbre sería de aproximadamente 228 horas (190 h + 20%).
